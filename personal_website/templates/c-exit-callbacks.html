Introduction

Unexpectedly exiting a C program can leave resources unreleased. The system handles this gracefully in some circumstances, not in others. This can lead to socket reuse errors, open filed descriptors, etc.

The standard library provides two built-in exit codes:

EXIT_SUCCESS 
#include <stdlib.h>

int main(void)
{
    // Terminate program and report succees
    exit(EXIT_SUCCESS);

    // Program never reaches here - exits above
    return 0;
}
EXIT_FAILURE 
#include <stdlib.h>

int main(void)
{
    // Terminate program and report failure
    exit(EXIT_FAILURE);

    // Program never reaches here - exits above
    return 0;
}
