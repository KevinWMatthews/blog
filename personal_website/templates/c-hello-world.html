What is this all about?

This series of posts is about good practices in C.

It has working examples of design patterns, testing strategies, and third-party libraries. There is a bias towards Linux and embedded systems, but using C is the primary focus.

Featured code samples can be compiled and run:

int main(int argc, char *argv[])
{
  printf("Hello, World!\n");
  return 0;
}

Some work is done behind the scenes to make this actually run, but you can find it all on GitHub.
