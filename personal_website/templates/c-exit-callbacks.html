Introduction

Unexpectedly exiting a C program can leave resources unreleased. The system handles this gracefully in some circumstances, not in others. This can lead to socket reuse errors, open filed descriptors, etc.

A program can exit with two (at least?) status codes,
success or failure:

#include <stdlib.h>

int main(void)
{
    // Report succees
    exit(EXIT_SUCCESS);

    // Alternatively, report failure
    // exit(EXIT_FAILURE);

    // Program never reaches here - exits above.
    return 0;
}
